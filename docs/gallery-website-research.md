## Boutique vs. Large Gallery Differences

Kay's Originals is a **boutique** gallery. Key traits to emphasize:
- Curated artist selection (5–30 artists, not thousands)
- Direct artist relationships highlighted
- About the gallery founder/owner is prominent
- Personal voice in copy (not corporate)
- More storytelling, less catalog
- Artist support/consignment focus
- Local community emphasis

Avoid: heavy filtering UI, community features, transaction-focused patterns.

## UI/UX Patterns Across High-Quality Gallery Sites

**Navigation:**
- Sticky dark/minimal nav bar visible always
- Logo left, links right
- Hamburger on mobile
- Search accessible (top right area)
- Breadcrumbs on detail pages

**Gallery Browsing:**
- 3–4 column grid desktop, 1–2 mobile
- Hover overlays showing artist name + title
- Filter/sort (by medium, artist)
- Large, high-quality artwork imagery
- Consistent aspect ratios per grid

**Artist Pages:**
- Hero portrait or signature artwork at top
- Bio/statement
- Portfolio grid
- Credentials / exhibition history

**Artwork Detail:**
- Large hero image (often full-width)
- Metadata: title, artist, date, medium, dimensions
- Artist link
- Related artwork

## Typography Trends (2025–2026)

- Serif headings: Georgia, Garamond, Playfair Display, DM Serif Display
- Clean sans-serif body: Outfit, Inter, Space Grotesk, system-ui
- Line height body: 1.5–1.7
- Line height headings: 1.1–1.2
- NO all-caps unless decorative accent use only

## Color Palette Trends

- **Luxury:** Black/charcoal + off-white + gold/brass
- **Contemporary:** Minimalist B&W + one jewel accent (navy, teal)
- Backgrounds: off-white (not pure #FFFFFF)
- Max 3–5 colors total
- High contrast text for readability

## Layout & Spacing

- Side margins: 40–80px
- Section vertical padding: 60–120px
- Generous whitespace around artwork ("breathing room")
- Alternating section backgrounds (off-white → light gray → white)

## Interactive Elements

| Pattern | Detail |
|---------|--------|
| Card hover | Lift (translateY) + shadow |
| Image overlays | Artist/title on hover |
| Scroll animations | Subtle fade-in on entry (not flashy) |
| Filter buttons | Highlight active state |
| Transitions | 200–300ms, ease |

> **Kay's preference:** Keep animations subtle and simple. No parallax, no dramatic entrances. Gentle fade-ins and hover lifts only. When in doubt, do less.

## What Makes a Site "Gallery-Quality"

1. High-quality imagery — no blurry or stretched images
2. Intentional whitespace — breathing room around content
3. Sophisticated typography — serif headings + clean sans-serif body
4. Limited palette — 3–5 colors including metallics (gold)
5. Smooth, subtle interactions
6. Mobile parity — equally beautiful on all devices
7. Clear hierarchy — easy to scan
8. Artist-first — artwork and artists are the hero, not UI chrome
9. Accessibility — readable fonts, high contrast, keyboard navigation
10. Editorial voice — storytelling, not product listings
