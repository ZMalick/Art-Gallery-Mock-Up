# Kay's Originals — Art Gallery Website

A mock art gallery website built for **CS 245: Working in a Team Environment**. Kay's Originals is a curated gallery celebrating bold, original artwork from visionary artists around the world.

## Live Preview

Open `index.html` in any modern browser to view the site.

## Project Structure

```
Kay's Originals Website/
├── index.html          # Homepage
├── css/
│   └── styles.css      # All styles (responsive)
├── js/
│   └── main.js         # Carousel, mobile nav, scroll animations
├── images/
│   └── KaysOrigLogo.png  # Site logo
└── README.md
```

## Features

- **Sticky Navigation** — Charcoal nav bar with dropdown menu for artwork categories (Paintings, Sculptures, Sketches), frosted glass effect, and mobile hamburger menu
- **Hero Carousel** — Full-width image slideshow with crossfade transitions, 10-second auto-rotation, prev/next arrows, dot navigation, and slide counter
- **Featured Artwork Grid** — Responsive CSS Grid displaying artwork cards with gold frame accents, hover overlays, and media tags
- **Browse by Category** — Three large image cards linking to Paintings, Sculptures, and Sketches
- **Artist Spotlight** — Artist profile cards with photos, bios, and portfolio links
- **Footer** — Logo, quick links, contact info, and copyright

## Design

- **Typography**: DM Serif Display (headings) + Outfit (body) via Google Fonts
- **Color Palette**: Deep Charcoal, Vivid Blue, Coral Red, Teal Green, Warm Gold
- **Responsive**: Desktop, tablet (≤1023px), and mobile (≤640px) breakpoints
- **Animations**: Scroll-reveal fade-ins with staggered delays

## Tech Stack

- HTML5, CSS3, vanilla JavaScript
- No frameworks or build tools required
- Images via [picsum.photos](https://picsum.photos) (placeholder service)
